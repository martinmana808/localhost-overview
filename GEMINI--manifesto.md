# LocalHost Overview

A premium macOS Menu Bar application that provides a real-time overview of all active localhost ports and the processes running on them. Use it to quickly identify which app is running where and navigate to them with a single click.

## Features
- Real-time port monitoring
- Process identification
- Click-to-open in browser
- Sleek, modern macOS interface
- Menu bar accessibility

## Tech Stack
- **SwiftUI**: For the native macOS user interface.
- **Process / Shell**: To query active ports using `lsof`.
- **AppKit**: For menu bar integration.
